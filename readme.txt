Create a stunning Android app of your wordpress blog

== Description ==

WPdroid is a wordpress plugin that allows you to create, in just a few clicks, stunning, feature-rich and presentable Android offline readers for your blog. You can share with the world by deploying on android market or hosting it by yourself.

Contact us: "appyet at gmail.com" for issue or support

== Installation ==

1. Upload `WPdroid` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
